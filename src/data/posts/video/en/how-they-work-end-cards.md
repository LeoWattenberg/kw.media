## Transcript

To create an end screen in YouTube, go to the Editor tab for the respective video. From there, you can create your own end screen with up to four elements or use one of the nine templates. You can also adjust the duration of the end screen and select which videos or playlists it should appear on. If you want to reuse an end screen from another video, you can import end screens from other videos.
